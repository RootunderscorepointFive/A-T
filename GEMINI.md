# A&T Group | Business Transformation Partners

## Overview
A&T Group is a South African accounting and advisory firm specializing in bookkeeping, IFRS reporting, tax, and strategic consulting.

## Tech Stack
- **Frontend:** HTML5, Vanilla CSS3, Vanilla JavaScript (ES6+).
- **Forms:** [Formspree](https://formspree.io/) for contact form handling.
- **Fonts:** DM Sans (Google Fonts).

## Project Structure
- `assets/`: Images, logos, and favicons.
- `css/`: Stylesheets.
- `js/`: Client-side scripts.
- `index.html`: Main entry point (single-page architecture).

## Conventions
- **Naming:** Kebab-case for file names and CSS classes.
- **Styling:** Mobile-first responsive design. IntersectionObserver used for reveal animations.
- **SEO:** Optimized with OpenGraph and JSON-LD schema for `FinancialService`.

## Deployment
- **Hosting:** Standard static hosting.
- **Sitemap:** [sitemap.xml](sitemap.xml)
- **Robots:** [robots.txt](robots.txt)
